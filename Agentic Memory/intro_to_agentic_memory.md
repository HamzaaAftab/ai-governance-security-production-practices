# Agentic Memory

- LLM are stateless. They don't have memory of the past interactions. 

- In AI agents, memory plays a very important role. The agent needs to remember the past interactions to provide relevant responses.

- Without memory, the agent would not be able to maintain the context of the conversation and would not be able to provide relevant responses.

- We will look at very broad and every kind of memory in Agentic AI. Memory itself is a very vast part in agentic systems.
